## Modules

The following table lists the primary modules included in the addon.

| Category     | Name               | Description                                                                                                      |
|:-------------|:-------------------|:-----------------------------------------------------------------------------------------------------------------|
| **Combat**   | OffhandPlus        | Advanced offhand with bind swap                                                                                  |
| **Player**   | AutoCraftPlus      | Automatically crafts items                                                                                       |
| **Player**   | AutoEnchantPlus    | Advanced shulker-based auto enchanting                                                                           |
| **Player**   | AutoReplenishPlus  | Automatically refills items in your hotbar or offhand                                                            |
| **Player**   | AutoSmithingPlus   | Advanced shulker-based auto smithing                                                                             |
| **Player**   | AutoTradePlus      | Automated villager trading system                                                                                |
| **Player**   | FakePlayerPlus     | Spawns a client-side fake player for testing usages                                                              |
| **Movement** | AutoJumpPlus       | Legit automatic jumping                                                                                          |
| **Movement** | BlinkPlus          | Delays movement packets to simulate lag                                                                          |
| **Movement** | ElytraUtils        | Various utilities for elytra flight                                                                              |
| **Movement** | NoSlowPlus         | Bypass Grim NoSlow checks                                                                                        |
| **Movement** | VelocityPlus       | Reduces incoming velocity effects                                                                                |
| **Render**   | DamageIndicator    | Displays damage dealt to entities                                                                                |
| **Render**   | LogoutSpotsPlus    | Logout with notifications                                                                                        |
| **Render**   | NewerNewChunks     | Detects new chunks by scanning the order of chunk section palettes (Note: Disabled if Trouser-Streak is present) |
| **World**    | AutoMountPlus      | Automatically mounts selected entities                                                                           |
| **World**    | AutoStripper       | Automatically strips wood and scrapes wax from copper                                                            |
| **World**    | SignScanner        | Scans, tracks changes, and saves sign text                                                                       |
| **Misc**     | AutoLoginPlus      | Login module based on server + nickname + password                                                               |
| **Misc**     | ChatControl        | Grants you full control over your chat experience                                                                |
| **Misc**     | ChatEncryptionPlus | Encrypt messages. Include msg encryption                                                                         |
| **Misc**     | HitSound           | Plays custom sounds on hitting or killing entities                                                               |
| **Misc**     | NBTTooltip         | Shows NBT data in item tooltips (Note: Disabled if Satellite is present)                                         |
| **Misc**     | RaidfarmAutomation | An raid farm automation tool. Made by NyaHub                                                                     |

## Features & Meteor Integrations

These features integrate directly into the Meteor Client configuration to provide additional functionality and audio feedback.

| Feature Name              | Description                                                                    |
|:--------------------------|:-------------------------------------------------------------------------------|
| custom-module-sound       | Plays custom sounds when toggling modules                                      |
| custom-chat-sound         | Plays custom sounds for different types of chat messages.                      |
| custom-visual-range-sound | Implements unique audio alerts for the Meteor visual range Notifier module     |
| custom-death-sound        | Triggers a specific audio clip upon player death                               |
| hide-meteor-icon          | Hide Meteor icon in chat. Find it in BetterChat                                |
| paused-on-kill-aura       | Pauses nuker when KillAura has a target. Find it in Nuker                      |
| paused-on-eat             | Pauses nuker when AutoEat is eating. Find it in Nuker                          |
| Sodium Occlusion Fix      | Fixes sodium occlusion                                                         |
| Air Place Fix             | Now you can use fireworks to fly, while Air Place enabled                      |
| Shulker Overview          | Overlays the most common item icon on shulker boxes. Find it in BetterTooltips |
